【MOD】SkySquids_1.3.1_v1.0.7

【概要】
空を飛ぶことができるイカを追加するMODです。
いわゆるイカフライです。
重力から解き放たれ、空に思いをはせたイカ･･･というイメージ。
微妙に重力の影響受けてますが。

高所でスポーンします。
クリエイティブモードのインベントリのスポーンエッグでイカフライを召喚できます。
イカのスポーンエッグと同じ色ですが、名前をよくみてご使用ください。

テイムできるかどうかはコンフィグで設定できます。
MMM氏のIKATORITame(イカトリテイム)のMODを導入していると自動的にテイムできます。(コンフィグで設定可能)

【コンフィグ】
・spawnLimitはロードしてるチャンク中の湧き上限

・IKATORITame未導入時
	canTame=false
	→テイムできない(default)
	canTame=ture
	→テイムできる
・IKATORITame導入時
	canTameWith=true
	→テイムできる(default)
	canTameWith=false
	→テイムできない
	
※困ったら全てfalseでテイムできない、全てtrueでテイムできる、です。
	
【特殊な動作】
基本的にIKATORITameと同じです。
・野生状態
	魚を持って右クリック
	→テイム
・テイム状態
	何も持たずに右クリック
	→頭に乗せる
	イカ墨を持って右クリック
	→上に乗る
・乗っている、または乗られている状態
	右クリック
	→おりる、またはおろす
	
魚を持っていると寄ってきます
